# **M8 – Readability-Agent**

## Ziel

Ein Agent, der Lesbarkeit bewertet (Klarheit, Stil, Verständlichkeit) und Ergebnisse im üblichen AgentResult-Format liefert.

## Aufgaben

* Kriterien für Lesbarkeit definieren.
* `ReadabilityAgent` implementieren (LLM-basiert).
* AgentResult-Struktur nutzen (score, errors, explanation).
* Integration in Pipeline, Postgres, Neo4j.
* Unit-Tests + Beispielcases.
